Android-Wifi-Scanner
====================


This is an [Android][1] app for scanning all available networks. It has been developed as a demo app.

----------

Features
--------

* Parameters shown:
 * SSID
 * BBSID
 * Channel
 * Signal strength
* Support for Android 2.2+
* Tested on Android KitKat 4.4.2



> **NOTE:**
>
> Use it under your own risk!
